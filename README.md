# Quantum-Weather

React + Vite weather webapp for Tsengel (Bayan-Ã–lgii). Uses Open-Meteo (no API key). Starry background, simple UI: only the brand text **Quantum** appears.

## Features
- 7-day forecast (daily) and hourly next-24h.
- Refresh interval (15m, 30m, 1h, 2h, 3h).
- Local caching (localStorage) for scale and offline-like behavior.
- Open-Meteo as data source (no API key).
- Deployable to Vercel / Netlify.
